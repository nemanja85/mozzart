# Mozzart


## 1.Clone the project

```sh
git clone git@github.com:nemanja85/mozzart.git
```
## 2. Navigate to the project directory

```sh
cd mozzart
```

## 3. Install the required dependencies

```sh
npm install
```

# Running the Project
### To start the project, use the following command:

```sh
npm run dev
```
